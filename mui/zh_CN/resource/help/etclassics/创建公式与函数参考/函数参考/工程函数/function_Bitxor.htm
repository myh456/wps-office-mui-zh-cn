BITXOR

返回两个数的按位“异或”。

　

语法

BITXOR( number1, number2)

number1  必须为十进制格式并大于或等于 0。

number2  必须为十进制格式并大于或等于 0。

　

说明

  • BITXOR 返回一个十进制数。
  • 如果任一参数小于 0，则 BITXOR返回错误值 #NUM! 。
  • 如果任一参数是非整数或大于 (2^48)-1，则BITXOR返回错误值 #NUM! 。
  • 如果任一参数是非数值，则 BITXOR返回错误值 #VALUE! 。
