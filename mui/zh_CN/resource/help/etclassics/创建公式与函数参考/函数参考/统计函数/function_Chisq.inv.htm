CHISQ.INV

返回 χ2 分布的左尾概率的反函数值。

　

语法

CHISQ.INV(probability,deg_freedom)

Probability   与 χ2 分布相关联的概率。

Deg_freedom   自由度数。

　

说明

  • 如果参数是非数值型，则 CHISQ.INV 返回 #VALUE! 错误值。
  • 如果 probability < 0 或 probability > 1，则 CHISQ.INV.RT 返回 #NUM! 错误值。
  • 如果 deg_freedom 不是整数，则将被截尾取整。
  • 如果 deg_freedom < 1 或 deg_freedom > 10^10，则 CHISQ.INV 返回 #NUM! 错误值。
