GAMMALN.PRECISE

返回伽玛函数的自然对数，Γ(x)。

语法

GAMMALN.PRECISE(x)

X     用于进行 GAMMALN.PRECISE 函数计算的数值。

说明

  • 若 x 为非数值型，则 GAMMALN.PRECISE 返回错误值 #VALUE!。
  • 若 x ≤ 0，则 GAMMALN.PRECISE 返回错误值 #NUM!。
  • 数字 e 的 GAMMALN.PRECISE(i) 次幂等于 (i-1)!，其中 i 为整数。
  • 函数 GAMMALN.PRECISE 的计算公式如下：

          GAMMALN.PRECISE = LN(Γ(x))
