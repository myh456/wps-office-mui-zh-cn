BINOM.INV 

返回一个数值，它是使得累积二项式分布的函数值大于等于临界值的最小整数。

语法

BINOM.INV(trials,probability_s,alpha)

Trials     独立试验的次数。

Probability_s     每次试验中成功的概率。

alpha   临界值，介于 0 和 1 之间，含 0 和 1 。

说明

  • 如果任意参数是非数值型，则 BINOM.INV 返回 #VALUE! 错误值。
  • 如果 trials 不是整数，将被截尾取整。
  • 如果 trials < 0，BINOM.INV 返回 #NUM! 错误值。
  • 如果 probability_s < 0 或 probability_s > 1，则 BINOM.INV 返回 #NUM! 错误值。
  • 如果 alpha < 0 或 alpha > 1，则 BINOM.INV 返回 #NUM! 错误值。
