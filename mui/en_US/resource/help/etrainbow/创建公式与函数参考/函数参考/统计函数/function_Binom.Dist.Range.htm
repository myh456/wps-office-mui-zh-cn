BINOM.DIST.RANGE 

使用二项式分布返回试验结果的概率。

语法

BINOM.DIST.RANGE(trials,probability_s,number_s,[number_s2])

Trials     独立试验次数。

Probability_s     每次试验成功的概率。 必须大于或等于 0 并小于或等于 1。

Number_s     试验成功次数。 必须大于或等于 0 并小于或等于 Trials。

Number_s2      如提供，则返回试验成功次数将介于 Number_s 和 number_s2 之间的概率。 必须大于或等于 Number_s 并小于或等于 Trials。

说明

    • 如果任何参数超出其限制范围，则 BINOM.DIST.RANGE 返回错误值 #NUM! 。
    • 如果任何参数是非数值，则 BINOM.DIST.RANGE 返回错误值 #VALUE! 。

=BINOM.DIST.RANGE(60,0.75,48)

返回二项式分布，即当成功概率为 75% 时，在 60 次试验之中出现 48 次成功的概率（0.084 或 8.4%）。
3 =BINOM.DIST.RANGE(60,0.75,45,50)

返回二项式分布，即当成功概率为 75% 时，在 60 次试验之中出现 45 到 50 次成功（包含）的概率（0.524 或 52.4%）。
